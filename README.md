# 42-charts-database

### INSTALATION

#### STEP 1 (node_modules)
$> `npm i`

#### STEP 2 (ENVIRONMENT VARIABLES)
In the projet root $> `touch .env`

then in your file add the folowing keys
```gradle
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=
DB_NAME=42Charts
```

#### STEP 3 (START)

$> `npm run scripts`
